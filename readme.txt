[Demo](https://www.a-j-evolution.com/plugins-wordpress/video-synchro-pdf/)

Lire une vid&eacute;o HTML5 et changer les pages d'un PDF en fonction de l'endroit o&ugrave; on est sur la vid&eacute;o, c'est maintenant possible !

Choisissez un PDF, une vid&eacute;o, d&eacute;finissez vos "timers" et le tour est jou&eacute; !

== Installation ==

L'installation du pluglin est tr&egrave;s simple.

1. Mettez le dossier 'videos-snychro-pdf' dans le dossier '/wp-content/plugins/' de votre site Wordpress
2. Activez-le dans la section 'Extensions' du menu WordPress
3. Un nouvel onglet appara&icirc;t dans la barre de menu &agrave; gauche dans votre interface back-office
